Kentang Goreng

Bahan:

  • Kentang
  • Garam
  • Tepung Maizena
  • Topping (opsional)

Cara Membuat:

  1. Potong kentang memanjang, lalu rendam dengan garam selama 1-3 jam
  2. Cuci bersih kentang. Kemudian Rebus hingga matang (5-6 menit)
  3. Tiriskan sampai tidak ada air. Tambahkan tepung maizena dan balur sampai rata
  4. Goreng dengan api besar sampai cukup keras, lalu kecilkan api dan goreng lagi 2 menit
  5. Masukkan freezer jika ingin dibuat stok
  6. Goreng lagi dengan api besar sekitar 1 menitan hingga matang
  7. Tiriskan
  8. Tambahkan topping sesuai selera
